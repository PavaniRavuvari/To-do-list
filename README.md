# To-Do List Web App

## Description
A modern and interactive To-Do List web application built using HTML, CSS, and JavaScript.  
The application allows users to manage daily tasks efficiently with a clean user interface and animated background representing everyday activities.

---

## Features
- Add new tasks  
- Delete tasks  
- Interactive and user-friendly UI  
- Animated background with daily activity icons  
- Responsive and modern design  
- Lightweight and fast (no external libraries)

---

## Technologies Used
- HTML5  
- CSS3 (Animations, Glassmorphism, Gradients)  
- JavaScript (DOM Manipulation)

---

todo-list-app/

│── index.html

---

## Use Case
This application helps users organize and track their daily activities such as studying, working, exercising, and personal tasks, improving productivity and task management.
